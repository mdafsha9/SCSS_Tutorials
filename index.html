Syntactically Awesome Style Sheets

Variables

Variables as a way to store information that we want to reuse through out stylesheet.
We can store things like color, background-color, font-stacks and any CSS value that we want to reuse.
Sass use $ symbol to make something a variable.

Variables example

$font-stack: Helvetica, sans-serif;
$primary-color: #333;
body{
font:100% $font-stack;
color:$primary-color;
}
